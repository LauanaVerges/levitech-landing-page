# LeviTech — Página inicial

Projeto front-end da primeira página do site LeviTech, desenvolvido com:

- HTML5 semântico
- CSS3 responsivo
- JavaScript puro
- Lucide Icons via CDN
- Google Fonts: Inter e Rajdhani

## Como abrir

1. Extraia a pasta do projeto.
2. Abra o arquivo `index.html` no navegador.
3. Para desenvolvimento, recomenda-se usar a extensão **Live Server** no VS Code.

## Estrutura

- `index.html`: estrutura completa da página.
- `style.css`: layout, cores, responsividade e animações.
- `script.js`: menu mobile, busca, carrinho demonstrativo, carrossel e animações.
- `assets/`: logo e mascote tratados para uso no site.
- `referencia-layout.png`: imagem de referência fornecida.
